Directly copying the results of a command to the clipboard

When you’re moving back and forth between a command line and another document, you may wish to run a command and then copy its results to the clipboard for use in the document or another location. To facilitate this, DTerm can quickly copy the entire results currently displayed to the clipboard.

To copy the currently visible results to the clipboard:

Choose “Copy results” from the action (gear) popup.

If you use the keyboard shortcut for the popup menu item, this can save your hand the round trip between keyboard and mouse to select the results, speeding up your workflow.
