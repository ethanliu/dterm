A command gives a message like “error opening terminal” or “standard input is not a tty”

DTerm is not an interactive terminal, so commands which require one may fail. You are most likely to encounter this with text editors like emacs, vi, pico, and nano.

To use commands requiring an interactive terminal:

Use DTerm's “Execute in Terminal” command instead of directly executing in DTerm. See Running commands in Terminal.
