Inserting selected items into the command line

When you’re moving back and forth between a command line and another document, it’s often the case that you are running commands with the document file as an argument. To facilitate this, DTerm can insert the filenames of the document(s) you are working on into the command for you.

If a Finder window was frontmost when DTerm was activated, DTerm’s selected items list is pulled directly from that window. Otherwise, if another application’s window was frontmost when DTerm was activated, the document file represented by that window is considered the one and only selected item.

To insert the list of selected files into the current command:

Choose “Insert selected items” from the action (gear) popup.
