How can I change the keyboard shortcut used to activate DTerm?

A system-wide keyboard shortcut is used to open a DTerm window regardless of the application you are currently using. Other such “hotkeys” are used, for instance, to log the current user out, activate Exposé or Dashboard, or to open the Spotlight window. Many of these may be seen via Apple Menu > System Preferences... > Keyboard & Mouse > Keyboard Shortcuts.

You may wish to change the keystroke used by DTerm because another application is using the same hotkey, or simply to better suit your personal keyboard usage.

To change the keyboard shortcut used to activate DTerm:

Choose DTerm > Preferences... from the menu bar.

Select the “General” preferences tab, if not already selected.

Click in the hotkey control next to “Activation hotkey:”, and type your desired new keystroke.
